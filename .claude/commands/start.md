# kintone アプリ作成ワークフロー開始

kintoneアプリを対話形式で作成するワークフローを開始します。

## 実行内容

`/kintone-workflow` スキルを実行してください。

このワークフローでは以下のフェーズを順番に実行します：

1. **Phase 1（概要書生成）**: 業務要件を収集し、システム概要書を作成
2. **Phase 2（設計書生成）**: 概要書からkintoneアプリ・フィールドを設計
3. **Phase 3（デプロイ）**: 設計書に基づいてkintoneにアプリを作成
4. **Phase 3.5（カスタマイズ適用）**: パターンベースでJavaScript/CSSを生成・適用

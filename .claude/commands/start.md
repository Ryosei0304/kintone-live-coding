# kintone アプリ作成ワークフロー開始

kintoneアプリを対話形式で作成するワークフローを開始します。

## 実行内容

`/kintone-workflow` スキルを実行してください。

このワークフローでは以下のフェーズを順番に実行します：

1. **Phase 1（コンテキスト把握）**: ペインポイント起点でWHO/WHYを整理
2. **Phase 2（業務フロー整理）**: As-Is/To-Beフロー設計、ビジネスイベント洗い出し、カスタマイズ提案
3. **Phase 3（アプリアーキテクチャ）**: ビジネスイベントからアプリ境界を決定、ER図生成
4. **Phase 4（統合レビュー + 詳細設計）**: クロスバリデーション、アプリ・フィールド設計書生成
5. **Phase 5（デプロイ + フィードバック）**: kintoneへデプロイ、カスタマイズ適用、フィードバック収集
